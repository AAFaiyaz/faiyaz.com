# Personal Resume Website

React + TypeScript + Vite portfolio/resume website for Abdullah Al Faiyaz.

## Run locally

```bash
npm install
npm run dev
```

## Add your profile image

Put your photo here:

```bash
public/faiyaz.jpg
```

The website already uses:

```tsx
<img src="/faiyaz.jpg" alt="Abdullah Al Faiyaz" />
```

## Update personal links

Open `src/main.tsx` and replace:

- phone
- WhatsApp link
- LinkedIn URL
- GitHub URL
- project GitHub links
- missing project live links

## Build

```bash
npm run build
```
